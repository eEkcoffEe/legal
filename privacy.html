Политика конфиденциальности

Настоящая политика определяет порядок обработки персональных данных пользователей сервиса.

Какие данные собираются

  • Telegram ID
  • Имя пользователя Telegram
  • Информация, необходимая для оказания услуг
  • Данные об оплате (без хранения банковских карт)

Для чего используются данные

  • оказание услуг;
  • обработка платежей;
  • техническая поддержка;
  • улучшение качества сервиса.

Передача третьим лицам

Данные могут передаваться платежным системам исключительно для обработки платежей.
